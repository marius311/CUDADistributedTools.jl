CUDADistributedTools.jl

A set of utility tools for multi-GPU + multi-process workflows with CUDA.jl + Distributed.jl.

      CUDADistributedTools.assign_GPU_workers — Method
      assign_GPU_workers(;
    print_info = true, 
    remove_oversubscribed_workers = false
    master_has_own_gpu = false, 
)

      Assign each Julia process a unique GPU. Assignment is done using CUDA.device!. Works with workers which may be distributed across different hosts, and with each host possibly having multiple GPUs.

      Keyword arguments

      • remove_oversubscribed_workers — If true, remove worker processes for which there are no free GPUs left. If false, an error is raised instead in this case.

      • master_has_own_gpu — Controls whether the master process counts as using its GPU, or whether a worker is free to also be assigned the same GPU.

      • print_info - Print CUDADistributedTools.proc_info after assignment.

      Example
